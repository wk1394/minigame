# 模块说明

## 文件结构

```
app.py              - 主应用入口（6.4KB）
├── 登录注册功能
├── 游戏路由注册
└── 页面路由

user.py             - 用户管理模块（5.0KB）
├── 用户列表
├── 添加用户
├── 用户积分
└── 会员信息

words.py            - 字字大冒险模块（3.6KB）
├── 词库管理（增删改查）
├── 游戏数据生成
└── 管理后台

multiplication.py   - 乘法消消乐模块（0.7KB）
└── 预留扩展接口
```

## 模块职责

### app.py - 主应用
- 注册所有蓝图模块
- 提供登录/注册功能
- 管理所有页面路由
- 配置游戏列表

### user.py - 用户管理
- 用户列表查询
- 用户添加
- 积分管理
- 会员信息管理

### words.py - 字字大冒险
- 词库 CRUD 操作
- 游戏数据生成
- 词库验证
- 管理界面

### multiplication.py - 乘法消消乐
- 预留后端接口
- 游戏配置（待扩展）

## 启动方式

```bash
python app.py
```

访问: http://localhost:3002
